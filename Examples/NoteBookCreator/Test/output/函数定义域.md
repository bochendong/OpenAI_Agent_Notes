# 函数的定义域
## 介绍

本章将介绍函数定义域的基本概念、判定方法及相关性质。通过定义、定理与例题，帮助读者系统掌握如何分析一个函数的定义域、常见的判定陷阱以及实际应用。
## 定义

函数的定义域是指使函数有意义的自变量x的所有取值的集合，通常用符号D表示。也就是说，定义域是所有能够代入函数表达式并获得有意义函数值的x的集合。
## 定理

### 定理 1

设函数 f(x) 在某集合 S 上的表达式为代数式、分式、根式、对数式、三角函数等，则函数 f(x) 的定义域为使得表达式有意义的所有实数 x 构成的集合。具体地：
1. 如果含有分母，则分母不为零；
2. 如果含有偶次根（如 sqrt、四次方根），则根号内表达式非负；
3. 如果含有对数（如 log_a(g(x))），则真数大于零且底数大于零且不等于 1；
4. 如果含有三角函数，则考虑其定义域（如 tan(x) 时，x ≠ π/2 + kπ）。
函数定义域被看作是所有使函数表达式有意义的 x 的全体。

**证明：**

证明（分析和求法步骤）：
1. 首先列出 f(x) 的表达式，分析其中包含哪些影响定义域的运算（如分母、根号、对数、三角函数）。
2. 对于每一种情况：
   - 分母：令分母 ≠ 0，解关于 x 的不等式。
   - 偶次根式：令被开方数 ≥ 0，解关于 x 的不等式。
   - 对数式：令真数 > 0，解关于 x 的不等式；同时保证底数 > 0 且底数 ≠ 1。
   - 反三角函数等特殊函数，也要保证其运算合法。
3. 将所有条件联立，取它们的交集，得到定义域。

举例说明：
设 f(x) = sqrt(x-1) / (x^2-4) + log_2(x-1)
(1) sqrt(x-1) 有定义，要求 x-1 ≥ 0，即 x ≥ 1
(2) 分母 x^2-4 ≠ 0，即 x ≠ 2, x ≠ -2
(3) log_2(x-1) 有定义，x-1 > 0，即 x > 1
由 (1) 和 (3)，x > 1；再结合 (2)，x ≠ 2
故定义域为：x ∈ (1, 2) ∪ (2, +∞)

因此，函数定义域的本质是由所有幂、分式、根式、对数等表达式的有效性条件共同决定的所有 x 的集合。

## 练习题

### 第 1 题

**题目：** 什么是函数的定义域？下列选项中哪个最准确地描述了函数的定义域的含义？

**选项：**
- A. 所有使函数有意义的自变量的取值集合
- B. 所有使函数值等于零的自变量集合
- C. 所有函数值的集合
- D. 任意实数集合

**正确答案：** A

**解释：** 函数的定义域是指所有使函数表达式有意义的自变量的取值集合。选项A准确地描述了这一点。选项B混淆了定义域和零点；选项C描述的是值域；而选项D过于宽泛，对于不同的函数并不总是所有实数都为定义域。

### 第 2 题

**题目：** 设集合A = {1, 2, 3, 4}，B = {2, 3, 5}，定义函数f(x)仅在x属于A且x在B中的时候有定义。请写出此函数的定义域：[空1]。

**答案：**
- 空1: {2, 3}

**解释：** 函数f(x)的定义域为A与B的交集，即A ∩ B = {2, 3}。

### 第 3 题

**题目：** 下列哪个运算可以用来描述“函数f(x)的定义域是集合A和B的交集”？

**选项：**
- A) A和B的并集
- B) A和B的交集
- C) A减去B
- D) B减去A

**正确答案：** B

**解释：** 正确答案是B。函数的定义域有时需要同时满足两个条件，这时定义域就是满足条件A的集合和满足条件B的集合的交集。交集指的是同时属于集合A和集合B的所有元素，这正好描述了定义域为A和B的共同部分。选项A“并集”指属于A或属于B的所有元素，不符合“同时满足两个条件”的要求；C、D是集合的差集，分别表示属于A但不属于B或属于B但不属于A，也不符合题意。

### 第 4 题

**题目：** 已知函数f(x) = 1/√(x-1)，试写出其定义域，并简要说明理由。

**答案：** 函数的定义域为x > 1。

**解释：** 因为被开方数x-1要求大于0（即x-1 > 0），且分母不能为0，所以x > 1。

## 总结

本章节主要介绍函数的定义域的相关知识，并包括如下内容：

一、定义域的概念
- 定义域是指使函数有意义的自变量x的取值集合。
- 对于y=f(x)，定义域即所有使f(x)有定义（即能计算出结果）的x的集合。

二、确定函数定义域的一般方法
1. 分析表达式合法性：
   - 分母不能为零：若含有分式，需令分母不等于零。
   - 偶次根根号内大于等于零：如f(x)包含√g(x)，需令g(x)≥0。
   - 对数函数真数大于零：如f(x)=ln g(x)，需令g(x)>0。
   - 指数函数底大于零且不等于1：如a^x，a>0且a ≠ 1。
   - 其他特殊要求：如分段函数，按定义限制。

2. 联立多个要求求交集：表达式包含多个限制时，需同时满足所有条件。

三、常见类型函数定义域例举
- 一次函数、二次函数、多项式：定义域为全体实数R。
- 分式函数：去掉使分母为零的x。
- 根式函数：注意根号下不能为负。
- 对数、指数函数：按各自要求取值。

四、难点与易错点提示
- 多条件同时成立时，易遗漏某些限制。
- 对于分数求根式混合表达式，需综合考虑限制。

五、常见问题
Q: 为什么分母为零不能取？为什么根号下不能为负？为什么对数真数需大于零？
A: 分母为零时无意义；偶次根负数无实数解；对数真数≤0无定义。

六、学习建议
- 做题时要仔细检查表达式中的所有可能性。
- 善于用不等式和集合的方法求解定义域。

通过本章学习，学生应掌握各种常见函数的定义域判定方法，并能准确书写定义域。
